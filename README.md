# Digital Portfolio - Saiprasad Reddy Poola

A fully responsive, multi-page digital portfolio website showcasing skills, projects, and achievements.

## Features
- **Responsive Design**: Mobile-first approach, compatible with Tablets and Desktops.
- **Modern UI**: Clean aesthetic with cream/white color scheme.
- **Interactivity**: Smooth scrolling, hover effects, and mobile navigation.
- **Pages Included**:
  - Home
  - About
  - Skills
  - Projects
  - Achievements
  - Contact

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6)
- Font Awesome (Icons)
- Google Fonts (Poppins)

## Setup Instructions
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    ```
2.  **Open the project**:
    Navigate to the project folder and open `home.html` in your browser.

## Customization
- **Images**: Replace images in the `images/` directory.
- **CV**: Place your CV pdf in `assets/cv/` and update the link in `home.html`.
- **Colors**: Update CSS variables in `css/main.css` to change the color scheme.

## Deployment
This static site is ready to be deployed on:
- GitHub Pages
- Netlify
- Vercel
